# 📚 Oxc 学习路径 - 实用优先

> 从使用到原理，循序渐进地学习 Oxc

## 🎯 学习理念

- **先用起来** - 通过实际运行和使用理解功能
- **看得见的效果** - 每一步都有可运行的示例
- **逐步深入** - 从表层使用到底层原理
- **按需学习** - 根据兴趣选择深入方向

## 📖 三周学习计划

```
第一周: 工具链使用 (5-7 天)
  ├─ Day 1-2: Parser 基础
  ├─ Day 3-4: Linter 基础
  └─ Day 5-7: 其他工具初探

第二周: 核心概念 (7-10 天)
  ├─ 访问者模式
  ├─ AST 操作
  └─ Semantic 分析

第三周: 深入方向 (按兴趣选择)
  ├─ 方向 A: 贡献 Linter 规则
  ├─ 方向 B: 理解 Parser 实现
  ├─ 方向 C: 学习代码转换
  └─ 方向 D: 深入内存管理
```
